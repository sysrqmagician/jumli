JuMLi: Quick Lookup

Home GitHub Repository

From a Workshop URL

Visit https://jumli.sysrqmagician.dev/workshop/WORKSHOP_ID.
Example: To view the report for RimHUD, which can be found at the Steam Workshop under https://steamcommunity.com/sharedfiles/filedetails/?id=1508850027, visit https://jumli.sysrqmagician.dev/workshop/1508850027

From a Package ID

Visit https://jumli.sysrqmagician.dev/package/PACKAGE_ID.
Example: To view the report for RimHUD, which has the Package ID jaxe.rimhud, visit https://jumli.sysrqmagician.dev/package/jaxe.rimhud
